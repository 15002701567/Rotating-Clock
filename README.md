# Rotating-Clock
### A implementation of the popular  wallpaper
一款当下非常热门的手机壁纸——旋转时钟（罗盘时钟/八卦时钟）的网页版实现；<br/>
支持PC、平板、手机端，自适应屏幕大小；<br/>
采用HTML、CSS、原生JavaScript实现。<br/>
动态展示效果见 demo.mp4 <br />
